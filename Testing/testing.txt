#############1. Поиск билета без ввода пункта прибытия.

ШАГИ: 
1. Зайти на сайт и выбрать вкладку купить
2. В поле «Откуда» ввести город вылета (Минск)
3. Поле «Куда» оставить пустым
4. Выбрать тип «В одну сторону» или «Туда - обратно» (в одну сторону)
5. В поле «Дата туда» выбрать дату отправления (18.01.2020) 
6. Оставить количество пассажиров оставить по умолчанию (1 взрослый)
7. Нажать кнопку «Найти»
Ожидаемый результат: Поле ввода «Куда» выделиться красным цветом, при наведении на него высветится сообщение «’Куда’ обязательно для заполнения»
